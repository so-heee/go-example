# 標準ライブラリ

https://pkg.go.dev/std

- os        [ファイルやディレクトリの操作、環境変数の操作、プロセス情報の参照]
- tim       [時間]
- math      [計算]
- rand      [乱数]
- flag      [コマンドラインのフラグ]
- fmt       [文字列の入出力]
- log       [ログ]
- strconv   [文字列の型変換]
- strings   [文字列操作]
- bufio     [バッファ付き I/O]
- ioutil    [1.16から非推奨になった]
- regexp    [正規表現]
- sync      [同期処理]
- crypto    [暗号化]
- json      [json]
- sort      [ソート]
- context   [処理の締め切り・キャンセル信号]
